=== OVERVIEW ===
BarChart renders a vertical bar chart from an array of data points. Each bar's height is proportional to its value relative to the maximum value in the dataset. Bars can have individual colors or share a default color.

The component renders entirely with Box and Text primitives, so it works on all targets. An optional label row appears below the bars, and values can be shown above each bar. When barWidth is omitted, bars auto-size to fill the parent container evenly.

=== API / SYNTAX ===
Import: import { BarChart } from '@ilovereact/core';

BarChartBar type:
```ts
interface BarChartBar {
  label: string;
  value: number;
  color?: Color;
}
```

Usage:
```tsx
<BarChart
  data={[
    { label: 'Mon', value: 12 },
    { label: 'Tue', value: 28 },
    { label: 'Wed', value: 19 },
    { label: 'Thu', value: 35 },
    { label: 'Fri', value: 24 },
  ]}
  height={100}
/>
```

Props:

| Prop       | Type            | Default    | Description                                |
|------------|-----------------|------------|--------------------------------------------|
| data       | BarChartBar[]   | (required) | Array of bar data points                   |
| height     | number          | 120        | Chart area height in pixels                |
| barWidth   | number          | undefined  | Width of each bar (omit for auto-sizing)   |
| gap        | number          | 8          | Gap between bars in pixels                 |
| showLabels | boolean         | true       | Show label text below each bar             |
| showValues | boolean         | false      | Show value text above each bar             |
| color      | Color           | '#3b82f6'  | Default bar color (overridden per-bar)     |
| style      | Style           | -          | Outer container style overrides            |

=== PLATFORM NOTES ===
- BarChart uses Box and Text primitives exclusively. No canvas, SVG, or target-specific drawing APIs.
- When `barWidth` is omitted, the chart fills its parent width and distributes bars evenly with `flexGrow: 1`.
- When `barWidth` is specified, the chart width is computed as: `data.length * barWidth + (data.length - 1) * gap`.
- Each bar height is calculated as `Math.max(1, Math.round((bar.value / maxValue) * barAreaHeight))`, ensuring every bar is at least 1px tall.
- When `showValues` is true, 14px of vertical space is reserved above bars for value labels.
- Label text is `fontSize: 9`, `color: '#64748b'`. Value text is `fontSize: 9`, `color: '#94a3b8'`.
- Bars use `borderRadius: 3` for slightly rounded tops.
- When `data` is empty, an empty Box with the specified `height` is rendered.

=== CRITICAL RULES ===
- The `data` prop is required. An empty array renders an empty placeholder.
- Bar heights are relative to the maximum value in the data. If all values are equal, all bars render at full height.
- If `maxValue` is 0 (all values are 0), bars are normalized against 1, so they appear as minimum height (1px).
- Per-bar `color` overrides the component-level `color` prop for that specific bar.
- BarChart does not handle negative values. All values should be non-negative.
- When using `showValues`, ensure adequate `height` to accommodate both bars and value labels (reserve 14px for values).
